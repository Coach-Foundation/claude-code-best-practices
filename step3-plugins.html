Install Recommended Plugins

You're still in your Claude Code session. Type each of these one at a time, pressing Enter after each:

/install-plugin superpowers
Adds structured workflows for planning, debugging, and code review. Helps you work faster with built-in superpowers.
/install-plugin context7
Gives Claude access to the latest library and framework documentation. Never worry about stale docs again.
/install-plugin code-simplifier
Reviews your code for quality and efficiency. Catches common issues and suggests improvements.
These plugins are optional but highly recommended. They extend Claude Code's capabilities and are free to install.
